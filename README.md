Local RAG AI Support

A fully local AI-powered customer support assistant built with:

  -Spring Boot

  -Ollama (Llama 3)

  -PostgreSQL + pgvector

  -Retrieval-Augmented Generation (RAG)


Features

  -Context-aware AI responses

  -Vector similarity search with pgvector

  -Knowledge base ingestion

  -Conversation logging

  -Fully local LLM execution (no OpenAI dependency)

  -Clean layered architecture
